# tpshop 电商平台 - 登录模块手动测试项目

## 项目介绍
- **项目背景**：tpshop电商平台（Web端）
- **测试模块**：账户登录页面（用户名/密码/验证码）
- **测试类型**：手动功能测试 + 测试用例设计 + Bug管理

## 测试内容
- 根据需求规格文档提取测试点（等价类划分法 + 边界值分析法 + 判定表）
- 设计并执行 **10个完整测试用例**（7列标准模板）
- 使用 **ZenTao** 提交 **3个真实Bug** 并跟踪
- 覆盖成功登录、用户名/密码/验证码异常等核心场景

## 项目成果
- 测试用例文档：`tpshop测试项目.xlsm`
- Bug报告截图：`bug.png`、`bug1.png`、`bug3.png`
- 工具：ZenTao、Excel

## 技术栈 / 技能点
- 软件测试基础理论（SDLC/STLC、黑盒测试）
- 测试用例设计（等价类、边界值、判定表）
- Bug生命周期管理 + ZenTao提Bug
- 文档规范编写
